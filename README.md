# Dashboard de Classificação de Íris

Dashboard interativo para análise exploratória de dados e classificação utilizando Árvore de Decisão.

## Funcionalidades
- Visualização de dados (dispersão, histogramas, mapa de calor de correlação)
- Filtragem interativa por espécie
- Modelo de Árvore de Decisão com acurácia e matriz de confusão

## Tecnologias
- Python
- Streamlit
- Scikit-learn
- Pandas
- Seaborn
- Matplotlib
